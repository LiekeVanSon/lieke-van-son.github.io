Day 2 of Hackathon: The effects of Rotation on stars

Today we are going to have a look at the effects of rotation on stars. We will explore three different cases that will have an effect on the the rotation of the star.

We will start from the test suite 15M_Dynamo in the /mesa/star/test_suite/ dir. Note how MESA has a lot of test suites available that can be very useful to start from when setting up a new model.

Similar to day one, we will start by copying the test suite to our home directory. 

	
	cp -r $MESA_DIR/star/test_suite/15M_dynamo/ ~/

note you might have to reload your JupyterLab environment to see the new folder in JupyterLab

If you open the folder in the Jupyter lab, you will see many different 'inlist' files (see day 1 for the introduction on these files). Go to https://docs.mesastar.org/en/latest/test_suite.html, and find the documentation for this test suite.

questions:

What is the purpose of the different inlist files?
Can you change the right inlist to activate the pgstar plot?

cd into the new folder created in your home directory and ./mk and ./rn MESA.

When you have successfully activated the pgstar plot, restart the run and a screen will pop up that looks like:

Example PGstar for the 15Msun dynamo model

Don't run the full test_suite, this will take about 40 min! Just end the run by typing Ctrl + C in the terminal, and go to the next section.
