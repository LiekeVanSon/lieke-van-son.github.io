Part 1: getting familiar with MESA

What is MESA?

MESA stands for Modules for Experiments in Stellar Astrophysics. It is a software package that can be used to simulate the evolution of stars. The introduction lecture slides can be found here.

'Using MESA' tutorial from the MESA website

We will start by following the instructions on the MESA website. Note that the text in "Running MESA" and "MESA Output" is mostly copied from the MESA website, though with slight modifications specific to the setup of this hackathon.
